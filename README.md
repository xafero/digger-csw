## Keyboard
| Label      | Description    |
| ---------- |:--------------:|
| Arrow keys | directional    |
| Control    | fire once      |
| Escape     | abort game     |
| Numpad +   | increase speed |
| Numpad -   | decrease speed |

* Microsoft Xbox One Controller
